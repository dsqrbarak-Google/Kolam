# מדריך התקנה - "קולם" בענן (Google Apps Script + GitHub Pages) 📱💻

## ⚠️ למה יש שגיאת מיקרופון? (הסבר טכני קצר)
גוגל חוסמת גישה לחומרה (מיקרופון ומצלמה) כאשר מריצים את האפליקציה ישירות דרך הקישור של ה-Web App של גוגל (`script.google.com`). הסיבה היא שגוגל טוענת את האפליקציה בתוך מסגרת פנימית מאובטחת (iframe) שאינה מאפשרת הרשאות מיקרופון.

## 💡 הפתרון (פשוט וקל):
נריץ את ממשק המשתמש (האתר) מתוך **GitHub Pages** (שהוא מאובטח, חינמי ומאפשר גישה מלאה למיקרופון), והאתר יתקשר ברקע עם השרת של גוגל!

---

## 🛠️ שלב 1: העלאת האתר ל-GitHub Pages (דקה אחת)
מכיוון שהקוד כבר נמצא בחשבון ה-GitHub שלך, כל מה שצריך זה להפעיל את האפשרות להציג אותו כאתר:
1. היכנס למאגר שלך ב-GitHub: `https://github.com/dsqrbarak-Google/Kolam`
2. לחץ על הלשונית **Settings** (הגדרות) למעלה.
3. בתפריט הצידי בצד שמאל, לחץ על **Pages**.
4. תחת **Build and deployment** -> **Source**:
   * ודא שרשום: **Deploy from a branch**.
   * תחת **Branch**, שנה מ-`None` ל-**`main`**.
   * השאר את התיקייה על **`/ (root)`**.
   * לחץ על כפתור **Save** (שמור).
5. המתן כדקה. רענן את העמוד, ותראה למעלה קישור ירוק שאומר:
   `Your site is live at https://dsqrbarak-google.github.io/Kolam/`
6. הקישור המדויק למסך הענן שלך הוא:
   **`https://dsqrbarak-google.github.io/Kolam/google-apps-script/index.html`**

## 🛠️ שלב 2: התקנת הקוד ב-Google Apps Script (פעם אחת)
1. היכנס לאתר: **[Google Apps Script](https://script.google.com/)**
2. לחץ על **New Project** (פרויקט חדש) ושנה את שמו ל-**`קולם`**.
3. לחץ על הקובץ **`Code.gs`** בעורך, מחק את הכל והדבק את הקוד מהקובץ **[Code.gs](Code.gs)** שבתיקייה זו. שמור (`Ctrl+S`).
4. לחץ על כפתור ה-**`+`** ליד Files בעורך, בחר ב-**HTML**, קרא לו בדיוק **`index`**, מחק את הכל והדבק את הקוד מהקובץ **[index.html](index.html)** שבתיקייה זו. שמור (`Ctrl+S`).
5. לחץ על כפתור ה-**Deploy** (פריסה) בפינה הימנית העליונה -> **New deployment**.
   * בחר בגלגל השיניים -> **Web app**.
   * **Execute as:** השאר על **Me** (האימייל שלך).
   * **Who has access:** שנה ל-**Anyone** (חובה כדי לאפשר לטלפון לתקשר עם השרת).
   * לחץ **Deploy**.
6. אשר את הגישה (לחץ על **Authorize Access** -> בחר באימייל שלך -> לחץ על קישור **Advanced** הקטן -> לחץ על **Go to קולם (unsafe)** -> ולחץ **Allow**).
7. בחלון שיופיע, תחת **Web app URL**, העתק את הקישור שנוצר (הוא מתחיל ב-`https://script.google.com/macros/s/...`).

---

## 🛠️ שלב 3: הפעלה ושימוש בנייד או במחשב
1. פתח בטלפון של אבא (או במחשב) את כתובת האתר שנוצרה בשלב 1:
   **`https://dsqrbarak-google.github.io/Kolam/google-apps-script/index.html`**
2. הדפדפן ישאל אותך מיד האם לאשר גישה למיקרופון -> **לחץ "אשר" (Allow)!**
3. לחץ על כפתור **⚙️ הגדרות מערכת** בפינה העליונה:
   * בשדה **קישור ה-Web App של גוגל** - הדבק את הקישור שהעתקת בסוף שלב 2 (הקישור של ה-Script).
   * בשדה **מפתח API של Gemini** - הדבק את מפתח ה-Gemini שלך.
   * בשדה **שם משתמש** - הקלד "אבא".
   * לחץ **שמור שינויים**.

המערכת תתחבר לענן, תשמיע הודעה קולית שהכל מוכן, ותתחיל לעבוד בצורה מושלמת מהנייד ומהמחשב! 🚀
